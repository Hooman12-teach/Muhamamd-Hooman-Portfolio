- **Tags**: Separate multiple tags with commas (e.g., "React, Node.js, MongoDB")
- **Featured Projects**: Only featured projects show on the homepage
- **Project Link**: Add a URL to make projects clickable
- **Data Storage**: All projects are saved in browser localStorage
- **No Backend Needed**: Everything works client-side!
